¿Qué es una función?
Una función es como una máquina automática que recibe un objeto o un número, le aplica siempre la misma regla fija y entrega un resultado nuevo.

¿Qué son los parámetros?
Los parámetros son los datos específicos que le das a una función para que pueda hacer su trabajo.

¿Qué hace return?
return es la acción que usa la función para entregar el resultado final y terminar su trabajo.

¿Qué diferencia hay entre Node y React?
Node y React son dos herramientas de JavaScript que hacen trabajos totalmente diferentes
React : Es una herramienta para construir lo que el usuario ve y toca en la pantalla (botones, menús, formularios)
Node : Es el entorno que permite a la computadora o al servidor ejecutar código JavaScript fuera del navegador.

¿Qué hace useState?
useState es una herramienta de React que le permite a un componente recordar información que puede cambiar mientras el usuario usa la aplicación.

¿Por qué usamos Number() en React?
Usamos Number() en React porque las cajas de texto de los formularios en las páginas web (los elementos <input>) siempre entregan la información en formato de texto (String), incluso si el usuario escribe un número.
